AVL – Supplier Evaluation

Every member of the Dream Team has gone through a rigorous qualifying process in order to be added to TCS’ Approved Vendor List (AVL). To make the AVL, an assessment is conducted at each manufacturing facility. Each plant is thoroughly evaluated, and must abide by industry standards & code regulations. This evaluation includes criteria in the following areas:

  • Front End Automation
  • Engineering Platform
  • In Process Flow
  • Preventative Maintenance Program
  • Equipment Calibration
  • Test Procedures
  • Final Inspection
  • Shipping/Job Tracking System
  • Scrap Percentage/ Returned Goods
  • Customer Complaint System

On-site inspections are then performed annually. To retain its status on the company’s AVL, each member must pass this perpetual supplier evaluation.
